# Sonwabile-AI-Assistant

## Overview
Sonwabile-AI-Assistant is an AI-powered workplace productivity platform designed to help professionals automate everyday work tasks using modern AI technologies.

The application combines multiple AI-driven tools into one integrated dashboard to improve productivity, communication, task management, and research efficiency.

This project was developed as part of the AI Skills Acceleration Programme (ASA_4) Week 4 Project.

---

## Features

### Smart Email Generator
- Generate professional emails instantly
- Multiple tone options:
  - Formal
  - Friendly
  - Persuasive
- Editable AI-generated responses

### Meeting Notes Summarizer
- Summarize lengthy meeting notes
- Extract:
  - Key discussion points
  - Action items
  - Decisions
  - Deadlines

### AI Task Planner
- Generate daily or weekly schedules
- Prioritize tasks intelligently
- Improve productivity and organization

### AI Research Assistant
- Summarize articles and topics
- Generate insights and recommendations
- Simplify workplace research tasks

### AI Chatbot Interface
- Interactive workplace AI assistant
- Responds to user prompts and questions
- Provides productivity support
